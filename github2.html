    # adding files to GitHub →

    # Do steps in this order →

       git clone https://github.com/kironroy/kironroy.github.io

       git add -A = add all files from computer folder to GitHub (Same for folders, but folders have to have content)
        or git add index.html = add one file at a time

       git commit -m "Add some notes about the commit" = notes/committing the files to GitHub

       git push origin master

    # removing files to GitHub →

    git rm -r one-of-the-directories

    git commit -m "Remove duplicated directory"

    git push origin master
